== Description ==

This plugin generates a fully customizable sitemap on your blog which helps search engines and archivers to index your web pages. 

The plugin supports multi-level categories and pages, exclusion of categories and pages, page navigation, permalink support, the option to choose what to display and tons of other cool features.

The HTML Sitemap Generator can also be used to offer your users an alphabetized index of your posts.
